Dieses Plugin ermöglicht die Konfiguration des Termine24-Online-Widgets auf
Ihrem Wordpress-Blog oder Ihrer Webseite.

== Description ==

Termine über Ihre Webseite erhalten - mit dem Termine24 Wordpress Plugin
können sie auf einfache Weise eine Terminvereinbarungsfunktion auf Ihrer
Webseite einbauen. Modern, und viel beliebter bei Ihren Kunden als ein
Kontaktformular.
Einfach das Plugin einbauen, einrichten und Termine erhalten.

Mehr Information zu Ihren Möglichkeiten mit Termine24 unter
www.termine24.de/partner

== Installation ==

Sie können genauso wie andere Plugins auf zwei Wegen installieren

Nr 1 - Widget herunterladen:

1. Zip-Datei herunderladen und entzippen. 
2. Den Ordner `termine24widget/` in den Ordner `/wp-content/plugins/`
hochladen.
3. Plugin über das WordPress-Menü "Plugins" aktivieren
4. Menüeintrag 'Design -> Widgets' im WordPress-Menü auswählen
5. Widget 'Termine24 Widget' auf die gewünschte Position ziehen
6. Ihre Termine24 ID - in den Widget-Einstellungen eingeben

Nr 2 - Über Ihr Wordpress-Backend:

1. Loggen Sie sich in Ihrem Wordpress-Backend ein
2. Menüeintrag 'Plugins -> Installieren' im WordPress-Menü auswählen
3. 'Termine24' im Suchfeld eingeben und 'Plugins suchen' klicken
4. Bei 'Termine24 Widget' auf 'Installieren' klicken und weiteren Anweisungen
folgen
5. Menüeintrag 'Design -> Widgets' im WordPress-Menü auswählen
6. Widget 'Termine24 Widget' auf die gewünschte Position ziehen
7. Ihre Termine24 ID - in den Widget-Einstellungen eingeben
